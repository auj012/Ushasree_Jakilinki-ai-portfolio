## How I work

- **Problem first, AI second.** I ask whether AI is the right solution before assuming it is — and I'm willing to park a project when the honest answer is "not yet."
- **Data honesty.** I check whether the data to build something actually exists before designing the model.
- **Evaluation is the product.** Guardrails, groundedness, precision/recall, explainability — the eval framework is where I put my energy.
